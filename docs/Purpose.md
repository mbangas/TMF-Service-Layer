
# Purpose

## What implements

This application fully implements the "Service Layer" within the TMForum Open Digital Architecture (ODA) / Framework;

In the TM Forum Open Digital Architecture (ODA), the Service Layer corresponds to the domain responsible for managing the service lifecycle (design, catalog, order, activation, assurance, testing, etc.).

It lies between:
- Product layer / Customer domain
- Resource / Network domain
